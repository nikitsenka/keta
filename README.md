# KETA - Knowledge Extract & Talk Agent

A multi-agent POC system for extracting knowledge from text documents into a graph database and chatting with the
extracted knowledge.

## Overview

KETA enables users to:

1. Define objectives for knowledge extraction
2. Upload text documents as sources
3. Extract entities and relationships into a knowledge graph
4. Chat with an AI agent that can query the extracted knowledge

## Key Features - POC Scope

- **Document Processing**: Upload and parse text documents
- **Knowledge Extraction**: Extract entities and relationships using LLM
- **Graph Storage**: Store knowledge in Apache AGE graph database
- **Knowledge Chat**: Ask questions and get answers based on extracted knowledge
- **Source Citations**: Track provenance of all extracted information
